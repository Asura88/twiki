## 使用答疑

1. 我应该为访问凭证配置什么权限？

    建议配置全局可读权限。

</br>

2. 未来的规划是什么？

    接下来会根据大家的反馈继续补充更多的云服务和云服务商，例如 RDS、域名等等。

</br>

3. 会支持 AWS、Azure 等国外的云吗？

    不打算支持国外的云，国外的云老外已经写过了，如果对这块儿有需求的，可以直接使用 [cloudlist](https://github.com/projectdiscovery/cloudlist) 这个工具。
